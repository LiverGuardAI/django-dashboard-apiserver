## Getting Started

### Using Docker
```bash
docker-compose up --build
```

### Manual Setup
```bash
cd reactproject
python manage.py migrate
python manage.py runserver
```

## API Endpoints
- Admin: http://localhost:8000/admin/
- API: http://localhost:8000/api/
